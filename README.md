## 本项目使用的第三方：
- Butter Knife：通过注解绑定View
- Frecro:加载圆角图片
- Retrofit2:网络请求
- Gson:JSON数据之间进行映射的java类库
- Litepal:数据库
- API接口：快递鸟
## 实现功能
- 查询快递状态
- 查询快递轨迹
